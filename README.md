# Bootstrap 5 Internship Project

## Project Overview
This project demonstrates the use of Bootstrap 5 components to build responsive web pages.

## Pages Included
- Home (Navbar, Hero, Cards, Footer)
- About Page
- Contact Page with Form

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5 (CDN)

## How It Works
Bootstrap is loaded using CDN. The grid system ensures responsiveness.
Forms and cards use built-in Bootstrap styling classes.
